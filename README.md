# Radio bot

> Created using `discord.js`

**How to use ?**

- Install required `node_modules`.
- Install `ffmpeg`.
- Add your prefix.
- Add your token.
- Add your embed color.
- Add your bot name.
- Add your inviteURL.
- Add your streamURL.
- Run `start.bat`.
