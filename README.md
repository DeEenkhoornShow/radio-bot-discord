# Radio bot

> Created using `discord.js`

**How to use ?**

- Install required `node_modules`.
- Install `ffmpeg`.
- Add your `prefix`.
- Add your `token`.
- Add your `embedColor`.
- Add your `botName`.
- Add your `inviteURL`.
- Add your `streamURL`.
- Run `start.bat`.
